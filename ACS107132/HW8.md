###一.

##在VirtalBox內建立一個新的Host-only 網路卡，網段為192.168.100.1/24

* 按右上角**全域工具**->**主機網路管理員**->**建立**
* 按**內容**調整剛剛建立的網路卡，選擇手動組態介面卡，將**IPv4 位址改為192.168.100.1**

![1]


##建立虛擬機器-1，並啟用host-only網路卡，透過ifconfig 或 ip指令，設定虛擬機器-1的網路為192.168.100.100/24

**建立虛擬機器(1)**
* 輸入` ip address add 192.168.100.100/24 broadcast + dev enp0s3 `設定1的網路

![2]



##建立虛擬機器-2，並啟用host-only網路卡，透過ifconfig 或 ip指令，設定虛擬機器-2的網路為192.168.100.200/24

**建立虛擬機器(2)**
* 輸入` ip address add 192.168.100.200/24 broadcast + dev enp0s3 `設定2的網路

![3]



##將二台虛擬機器的網路設定存至/etc/sysconfig/network-scripts/下相對應的ifcfg-*檔案，重新啟動虛擬機器，確認網路ip設定無誤。

#1

* 輸入` vi /etc/sysconfig/network-scripts/ifcfg-* `
> 按I進入編輯
1. 將**ONBOOT改成yes**
2. 新增IPADDR=192.168.100.200
> Esc->:wq儲存離開
> 重新開機

![4]

#2

* 輸入` vi /etc/sysconfig/network-scripts/ifcfg-* `
> 按I進入編輯
1. 將**ONBOOT改成yes**
2. 新增IPADDR=192.168.100.100
> Esc->:wq儲存離開
> 重新開機

![5]


##從虛擬機器-1 ping 虛擬機器-2確認網路是連通，並從虛擬機器-2 ping 虛擬機器-1，確認網路也是連通。

#1

* 輸入` ping -c 3 192.168.100.200 `確認連線
> 記得要加-c的參數限制執行的次數 **不然會發生跟我一樣的慘狀QQ**
> 真的忘記加了就按著ctrl跟c就可以出去了

![6]

#2

* 輸入` ping -c 3 192.168.100.100 `確認連線
