## HW2
#### 請依下列要求練習安裝Linux，並詳細記錄說明安裝過程。

1. 建立一個新的虛擬機器，distribution & image 任選。
+ 開啟VM，按下新增。
+ 設定虛擬器名稱，類型選擇Linux，再按建立。

![GITHUB]( https://imgur.com/bnLt98w.jpg"git圖示")

+ 硬碟檔案類型VDI，硬碟動態配置選擇動態配置。

![GITHUB]( https://imgur.com/d0UufUx.jpg"git圖示")

+ 啟動虛擬器。

![GITHUB]( https://imgur.com/smCGNDm.jpg"git圖示")

+ 選擇下載好的Centos 7虛擬光碟檔案。

![GITHUB]( https://imgur.com/rrUIhVL.jpg"git圖示")

+ 安裝Centos 7。

![GITHUB]( https://imgur.com/UOWPvtz.jpg"git圖示")

+ 選擇語言。

![GITHUB]( https://imgur.com/A2MxpX3.jpg"git圖示")

2. 至少建立二個分割區，分別掛載 / 與 /home。

+ 點安裝目的地，分割硬碟。

![GITHUB]( https://imgur.com/Ea2FFAS.jpg"git圖示")

![GITHUB]( https://imgur.com/O0fZFF8.jpg"git圖示")

+ 選取讓我自行配置磁碟分割，按完成。

![GITHUB]( https://imgur.com/wdqPDVp.jpg"git圖示")

+ 進入介面按下"+"新增分區"/"，並設置大小。

![GITHUB]( https://imgur.com/ukJRUW4.jpg"git圖示")

+ 同上，新增分區"/home"，並設置大小。

![GITHUB]( https://imgur.com/90SgdsD.jpg"git圖示")

+ 完成分割。

![GITHUB]( https://imgur.com/BbotrqZ.jpg"git圖示")

+ 安裝完成重啟。

![GITHUB]( https://imgur.com/9PqqnGk.jpg"git圖示")

3. 登入後用 df 驗證是否有掛載二個分割區。

+ 登入並用指令df檢查確認。

![GITHUB]( https://imgur.com/zXJbF7q.jpg"git圖示")

+ 完成。
