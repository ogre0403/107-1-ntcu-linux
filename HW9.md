##### 安裝軟體

### 閱讀 `https://www.cyberciti.biz/faq/installing-rhel-epel-repo-on-centos-redhat-7-x/` 後，依照其作法，啟用epel reposisitory，並安裝`htop`
1.  `yum install epel-release`
![GITHUB](https://imgur.com/8VQYMLB.jpg"git圖示")

2.  `sudo yum repolist`
![GITHUB](https://imgur.com/S2xK0xw.jpg"git圖示")

3.  `sudo yum search htop`
![GITHUB](https://imgur.com/axCThjg.jpg"git圖示")

4.  `sudo yum info htop`
![GITHUB](https://imgur.com/Yo7fVOZ.jpg"git圖示")

5. `sudo yum install htop`
![GITHUB](https://imgur.com/G0DYNpW.jpg"git圖示")
