第一題

由於CentOS7尚未內建wget。
所以先使用 指令yum -y install wget 安裝wget。
使用 wget 指令。

第二題

執行tar -czf var.tgz /var 2> tar-err.log 指令。
使用 cat tar-err.log 查看剛剛導出的錯誤。
