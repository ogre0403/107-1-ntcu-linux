1.開啟virtual box，點選右上方的全域工具，點進主機網路管理員，
再點選左上角的建立，建立新的網路卡後，將網段更改為192.168.200.100/24，
開啟虛擬機器，輸入yum install net-tools，來下載ifconfig，
下載好之後，輸入ifconfig enp0s8 192.168.200.100，再輸入ifconfig，
就可以找出目前有被啟動的網卡(圖1)。
-----------------------------------------------------------------------------
2.到網路上先下載Nginx檔，(我的載點:http://nginx.org/en/download.html)，
下載完成後，回到虛擬機器，輸入yum install nginx，就能完成安裝了(圖2)，
輸入systemctl start nginx.service，來啟動nginx，再輸入netstat -tlnp |grep ssh
，來確認連接到的port(圖3)。
-----------------------------------------------------------------------------
3.開啟瀏覽器，輸入192.168.200.102，成功連上網頁伺服器(圖4)。
-----------------------------------------------------------------------------
4.回到虛擬機器，輸入curl 192.168.200.100，會跑出html的程式碼，
就是連線成功到Linux上的nginx網頁伺服器了(圖5)。
------------------------------------------------------------------------------
5.輸入cat /var/log/nginx/error.log |cut -d ' ' -f 16 | sort | uniq -c，
可以看出錯誤的筆數(圖6)。
